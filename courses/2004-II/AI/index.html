Descripción del curso

Objetivo

Al finalizar el curso se espera que el estudiante conozca y aplique los fundamentos y algunas metodologías que hacen posible la inteligencia artificial, adquiera habilidades para determinar, describir y usar los principios técnicas que permiten el desarrollo de sistemas inteligentes.

Metodología

  • Los conceptos fundamentales de cada tema del curso se presentan y se aclaran en clase.
  • Se harán talleres en clase y fuera de ella que permitan afianzar el dominio de los tópicos discutidos.
  • Se realizarán proyectos con un componente considerable de programación que permitan aplicar las técnicas vistas en clase.
  • Se harán algunas evaluaciones individuales, escritas o prácticas, en las cuales los estudiantes deben mostrar un buen dominio de los conocimientos y habilidades que el curso pretende desarrollar.

Material de apoyo y recursos

  • Sitio web del libro guía [R&N]
    • Presentaciones
    • Implementaciones de algoritmos del libro en diferentes lenguajes
  • Programas ejemplo:
    • Ejemplo de algoritmos de búsqueda en Java usando la librería aima:
      • descomprima el archivo flipper.zip
      • cd flipper
      • javac -classpath aima.jar flipper/search/*.java
      • javac -classpath aima.jar flipper/gui/*.java
      • java flipper/gui/Application1
      • java -cp aima.jar flipper/search/Demo
      • Documentación disponible aquí
    • Servidor y clientes para el juego de Othello, instrucciones de ejecución:
      (versión actualizada Oct 7, corrección en documentación, arreglos en el servidor)
      • Revise la documentación del código.
      • Descomprima el archivo Othello.zip
      • En una consola, cambie al directorio Othello
      • Ejecute el servidor:

        java othellogame.ServidorGUI

      • Ejecute los clientes. Para cada cliente haga lo siguiente:
        1. abra una consola nueva y cambie al directorio Othello
        2. ejecute:

          java othellogame.Client localhost nombre

        3. Use un nombre diferente para cada cliente

        4. Si el servidor se encuentra en un computador diferente, use la dirección correspondiente en vez de localhost
        En la GUI del servidor haga los siguiente:
        1. Seleccione el primer jugador
        2. Seleccione el segundo jugador
        3. Presione el botón comenzar
        4. Presione el botón jugar sucesivamente hasta terminar el juego
