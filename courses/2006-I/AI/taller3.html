Taller 3

Algoritmos de Búsqueda no Informados

Entrega: Martes 18 de abril 2006 en clase
Grupos de máximo 3 personas
Inteligencia Artificial
2006-I

Objetivos

  • Hacer una implementación del algoritmo Búsqueda en Profundidad Iterada. 
  • Haga las modificaciones necesarias para que no se repitan estados en la búsqueda.
  • Usar el algoritmo para resolver el problema de las jarras:
    • El programa de solicitar la capacidad de cada una de las jarras y la cantidad a medir
    • El programa de retornar la solución, secuencia de acciones y estados, o imprimir 'no existe' si este es el caso.

Entregables

  • Archivo en HTML o PDF con:
    • Arquitectura global del programa: descripción clara y detallada de clases, funciones, etc
    • Ejemplos de prueba
    • Discusión
  • Código fuente debidamente documentado
  • Código objeto e instrucciones para correrlo

Notas:

  • Es posible que revisen otros códigos disponibles en Internet, pero la implementación la debe hacer cada grupo. 
  • Cada uno de los integrantes del grupo deben estar en capacidad de sustentar el trabajo.
  • El lenguaje de programación es libre.
  • Los talleres que cumplan con todos los requerimientos tendrán un bono especial.
